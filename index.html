Clarion Touch

Introduction

This repository provide examples for working with Touch in Clarion. There is a mix of .APP based projects and some are pure source files. We'll be adding additional examples so come back and visit the site, or clone the repository on your own machine.

We hope they'll help you get some ideas on supporting touch gestures in your Clarion programs. If you come up with some great ideas on how to make the demos better, just make a pull request!

Requirements: OS: Windows 8.x The support for touch is based on the API for Windows versions 8.x and above - and is untested on Windows 7 (and may not work completely)

Clarion: version 10.0 and up
